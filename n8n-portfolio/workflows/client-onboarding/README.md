# Client Onboarding Workflow

Automated end-to-end client onboarding pipeline with HubSpot CRM, Gmail drip sequence, and Telegram notifications.

## Flow
JotForm → Validate → HubSpot → Welcome Email → PDF Guide (2hrs) → VIP Offer (1 day) → Completion (2 days)

## Setup
1. Connect JotForm account and set your Form ID
2. Connect HubSpot OAuth2
3. Connect Gmail OAuth2
4. Connect Google Drive and update the PDF file URL
5. Set your Telegram Bot token and Chat ID
